## 📖 About the Project

The Cognitive Shadow Theory establishes **formal limits on the complete digitization of consciousness**. Any computational system claiming exhaustive formal representation of a cognizing agent inevitably leaves an unformalizable remainder — the cognitive shadow.

### Key Results

- **AUC = 0.947** for LIS vs healthy detection (GroupKFold)
- **AUC = 0.887** under cross‑population validation (Random Forest + SMOTE)
- **Specificity confirmed:** AUC ≈ 0.5 for control tasks (sleep, sex, epilepsy)
- **Parameter A (Awareness)** confirmed (p = 0.045)
- **Theorem 10:** AUC of any consciousness classifier ∈ (0.5, 1)
